Welcome

BrainStim v is a free/open source application.

BrainStim is a rich multiple document and various file type editor that has code editing features like automatic code completing and syntax highlighting.

BrainStim contains a automatic Plug-in loading framework that fully supports developers to easily write various extensions.

BrainStim contains an internal script engine whose environment can access a large part of the Qt SDK and loaded Plug-ins by making use of script bindings.

BrainStim is optimized for creating and running flexible behavioral experiments supported by the script engine and plug-in framework.

BrainStim has extensive support for creating behavioral experiments using fully integrated (Qt's) QML and optimized Retinotopic Mapping modules.

The application is currently only supported under Windows (32 and 64-bit architectures) but in the future BrainStim will be become cross-platform (Mac, Linux).
